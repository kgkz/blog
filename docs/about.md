# de-crypto.techについて
de-cryptoには以下の思いを込めました。  
### エンジニアの暗号を紐解く
エンジニア同士での会話は特殊な用語が錯綜することが珍しくありません。  
まるで暗号です。  
そんなよくわからない暗号たちを復号(decrypt)し、わかりやすく伝えたい。  

### 暗号資産を紐解く
個人的に興味のある暗号資産(crypto currency)を技術的に解読(decrypt)したい。
